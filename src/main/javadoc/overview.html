Here javadoc information the the Der Die Das software is provided.

Package overview

Here a summary of what contains each package is provided:
  • com.derdiedas - base package for the hole application. Contains only the starter class
  • com.derdiedas.authentication - Holds authentication and authorication classes
  • com.derdiedas.bootstrap - Holds system-wide initialization classes, such ones which populates the database with words.
  • com.derdiedas.bootstrap.importer - Holds classes for importing german words into the system
  • com.derdiedas.config.exception - Holds general configuration classes, such as cache and exceptions
  • com.derdiedas.controller - Holds controllers which exposes the REST API.
  • com.derdiedas.dto - Holds data transfer objects, used mostly for handling information in the REST API.
  • com.derdiedas.model - Holds data modeling classes. They reflect exactly the database model.
  • com.derdiedas.repository - Holds classes which interact with storage, mostly our database.
  • com.derdiedas.service - Holds the business logic classes.
